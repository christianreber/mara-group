# Mara Group Website

Static multi-page website prepared for Vercel deployment.

## Pages

- `/`
- `/properties`
- `/sell`
- `/houses/colonia-sant-pere`
- `/houses/belavent`

## Deploy

1. Push this folder to a GitHub repository named `mara-group`.
2. Import the repository in Vercel.
3. Framework preset: `Other` (static).
4. Build command: none.
5. Output directory: project root.

Vercel uses `vercel.json` rewrites to serve clean URLs.
